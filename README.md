

## **People Analytics Dashboard: Analisis Attrition & Performa Karyawan**

**Deskripsi Proyek**  
Dashboard analisis SDM komprehensif yang menganalisis pola attrition karyawan, metrik performa, dan faktor risiko burnout dari 100.000 data karyawan. Proyek ini menggabungkan preprocessing data berbasis Python dengan visualisasi interaktif Power BI untuk memberikan insights HR yang actionable.
